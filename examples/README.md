- l<sub>1</sub> = 1500 mm
- α = 30 deg
- EA = 1 x 10<sup>6</sup> N
- u<sub>app</sub> = 7 mm
- v<sub>app</sub> = 2 mm

### Solution
The problem is modelled as follows:

```
